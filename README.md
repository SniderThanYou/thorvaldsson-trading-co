# thorvaldsson-trading-co
It's more fun to trade with friends.

## setup

Create `secret.cfg`:

```
[BINANCE]
ACTUAL_API_KEY = 1234abcd
ACTUAL_SECRET_KEY = 5678defg
```

## entry points

- `backtest.py` to backtest a strategy
- `crypto_alerts.py` to run a live process, to alert you when cryptos meet certain conditions
